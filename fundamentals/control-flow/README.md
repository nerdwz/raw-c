# control-flow

> conditional logic in C using `if`, `else if`, `else`, and `switch`.

---

## what will be covered

- `if` — basic conditional execution
- `if / else if / else` — mutually exclusive branches
- nested `if` — conditions inside conditions
- `switch` — multi-case branching
- `case` grouping with fallthrough
- combining `if` and `switch`
- logical operators: `&&`, `||`, `!`
