# challenges — io-basics

> problems to practice printf and scanf.

---

## challenge 01 — personal card

**Problem:**
Write a program that asks the user for their name, age, and height.
Then print a formatted personal card with all the information.

**Expected output:**
```
=== Personal Card ===
Name   : John
Age    : 22
Height : 1.75m
```

**File:** `challenge01.c`

---

## challenge 02 — simple calculator

**Problem:**
Ask the user for two numbers and an operation (`+`, `-`, `*`, `/`).
Print the result of the operation.

**Expected output:**
```
Enter two numbers: 10 5
Enter operation: +
Result: 10 + 5 = 15
```

**File:** `challenge02.c`
