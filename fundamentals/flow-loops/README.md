# flow-loops

> iteration and loop control in C using `for`, `while`, `do while`, `break`, and `continue`.

---

## what will be covered

- `for` — counter-controlled iteration
- `while` — condition-controlled iteration
- `do while` — execute-then-check loop
- `break` — exit a loop early
- `continue` — skip to the next iteration
- nested loops — loops inside loops
- loop patterns — building shapes and sequences
- accumulators — summing and counting with loops
- input validation — looping until valid input
